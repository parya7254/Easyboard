# Easyboard
A hot swappable tenkeyless (full size but without the number pad) mechanical keyboard with RGB lighting and an OLED (SSD1306) display! It will use a RP2040 Pro Micro as the controller, SK6812-MINI-E (reverse mount) LEDs for the backlight, and MX-style switches (and also MX-style hot swap sockets).

# Why did I Make This??
I really like designing my own things, like REALLY. And I also REALLY want a mechanical keyboard. So I thought to myself, *why not just make your own?* So I did one of my most complex projects ever, designing and building my own hot-swappable keyboard with RGB lights. And after about *more than* 60 **MIND GRUELING** hours of work, I managed to finally make a mechanical keyboard, one that was made by **me** and **me** only. And another reason for making this keyboard is that I have a lot of school work which requires a lot of typing and sometimes I write a lot of code, so this keyboard will help me work more efficiently.

The hardest part of designing this keyboard was wiring the things together in my PCB, expecially the LEDs. I ran into many rabbit holes during this process since lots of things were in the way and there was not encough space to wire some things so I had to move some traces and sometimes entirely rewire some things just so I can wire other things. Designing the keyboard made me want to give up and scrap everything and start another project, but I still continued and managed to finish everything.
