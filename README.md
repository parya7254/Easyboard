# Easyboard
A hot swappable tenkeyless (full size but without the number pad) mechanical keyboard with RGB lighting and an OLED (SSD1306) display! It will use a RP2040 Pro Micro as the controller, SK6812-MINI-E (reverse mount) LEDs for the backlight, and MX-style switches (and also MX-style hot swap sockets).
